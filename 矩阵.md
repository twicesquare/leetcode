# 反转或90°等
| 目标效果 | 核心公式 | 逻辑拆解 |
| :--- | :--- | :--- |
| 纯转置 | `[list(r) for r in zip(*matrix)]` | 行列互换 |
| 顺时针 90° | `[list(r[::-1]) for r in zip(*matrix)]` | 转置 + 左右翻转 |
| 逆时针 90° | `[list(r) for r in zip(*matrix)][::-1]` | 转置 + 上下翻转 |
| 旋转 180° | `[r[::-1] for r in matrix][::-1]` | 左右翻转 + 上下翻转 |
