# 前、中、后序的递归写法
```
class Solution:
    def preorderTraversal(self, root): # 前序：根 -> 左 -> 右
        if not root: return []
        return [root.val] + self.preorderTraversal(root.left) + self.preorderTraversal(root.right)

    def inorderTraversal(self, root):  # 中序：左 -> 根 -> 右
        if not root: return []
        return self.inorderTraversal(root.left) + [root.val] + self.inorderTraversal(root.right)

    def postorderTraversal(self, root): # 后序：左 -> 右 -> 根
        if not root: return []
        return self.postorderTraversal(root.left) + self.postorderTraversal(root.right) + [root.val]
```
递归写法完全对应了遍历的定义。三种遍历的代码结构几乎一模一样，只是记录值的顺序不同。

# 前、中、后序的递归写法（使用栈）
在面试中，面试官经常要求不使用递归，因为递归在树极深时会导致栈溢出（Stack Overflow）。
## 前序遍历（最简单）
利用栈“先进后出”的特性。为了让左孩子先出栈，我们要先压入右孩子。
```
def preorder(root):
    if not root: return []
    stack, res = [root], []
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.right: stack.append(node.right) # 右孩子先入
        if node.left: stack.append(node.left)   # 左孩子后入，下次先弹
    return res
```
## 中序遍历（最常用）
模拟“一路向左”直到尽头的过程
```
def inorder(root):
    res, stack, curr = [], [], root
    while curr or stack:
        while curr:          # 一直往左扎到底
            stack.append(curr)
            curr = curr.left
        curr = stack.pop()   # 弹出的就是当前最左的“根”
        res.append(curr.val)
        curr = curr.right    # 转向右子树
    return res
```
## 后序遍历（反转小技巧）
直接写后序迭代较难，但我们可以观察：

前序：根 $\rightarrow$ 左 $\rightarrow$ 右

变形：根 $\rightarrow$ 右 $\rightarrow$ 左

反转变形结果：左 $\rightarrow$ 右 $\rightarrow$ 根（即为后序！）

```
def postorder(root):
    if not root: return []
    stack, res = [root], []
    while stack:
        node = stack.pop()
        res.append(node.val) # 顺序：根 -> 右 -> 左
        if node.left: stack.append(node.left)
        if node.right: stack.append(node.right)
    return res[::-1] # 反转，得到 左 -> 右 -> 根
```
