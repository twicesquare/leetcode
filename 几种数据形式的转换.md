# list转字典
## 1. 两个列表合并（一个做 Key，一个做 Value）
如果你有两个长度相同的列表，想把它们一一对应起来，zip 函数是最高效的选择。
```
keys = ['name', 'age', 'city']
values = ['Alice', 25, 'New York']

# 使用 zip 将两个列表打包，再转成 dict
res = dict(zip(keys, values))

print(res)
# 输出: {'name': 'Alice', 'age': 25, 'city': 'New York'}
```

## 2. 将“键值对”列表直接转换
如果你的列表本身就是嵌套结构，且每个子项刚好有两个元素（如 `[key, value]`），可以直接调用 `dict()`。
```
pairs = [['a', 1], ['b', 2], ['c', 3]]

res = dict(pairs)

print(res)
# 输出: {'a': 1, 'b': 2, 'c': 3}
```

## 使用“字典推导式”（最灵活）
### 场景 A：把索引当作 Key，元素当作 Value
```
fruits = ['apple', 'banana', 'cherry']
res = {i: fruits[i] for i in range(len(fruits))}
# 输出: {0: 'apple', 1: 'banana', 2: 'cherry'}
```

### 场景 B：把元素当作 Key，它的长度或某种属性当作 Value
```
fruits = ['apple', 'banana', 'cherry']
res = {f: len(f) for f in fruits}
# 输出: {'apple': 5, 'banana': 6, 'cherry': 6}
```

## 4. 统计频率`Counter`
如果你想统计列表中每个元素出现的次数，可以使用 `collections.Counter`，它本质上就是一个字典。
```
from collections import Counter

nums = [1, 2, 2, 3, 3, 3]
res = dict(Counter(nums))

print(res)
# 输出: {1: 1, 2: 2, 3: 3}
```

## 5. 将列表中的对象按某个属性分组
假设你有一个包含多个字典（或对象）的列表，你想通过某个唯一标识（如 id）来快速查找它们：
```
users = [
    {'id': 101, 'name': 'Alice'},
    {'id': 102, 'name': 'Bob'}
]

# 以 id 为键，整个字典为值
user_dict = {u['id']: u for u in users}

print(user_dict[101])
# 输出: {'id': 101, 'name': 'Alice'}
```
## 避坑

键必须唯一：如果列表中存在重复的键，后面的值会覆盖前面的值。

键必须不可变：就像我们之前讨论 set 时提到的，列表不能作为字典的键，必须先转成元组（Tuple）。

## 2、创建二维全 1 数组
如果你需要一个 $m \times n$ 的矩阵：

纯 Python (切记不能直接 `[[1]*n]*m`)：

直接乘会导致每一行都指向同一个内存地址，改一个全都会变。必须使用列表推导式。

```
m, n = 3, 4
ans = [[1] * n for _ in range(m)]
```
在 Python 的列表推导式（List Comprehension）中，`_`（下划线）是一个约定俗成的占位符。

它的核心含义是：“我需要循环这么多次，但我并不关心当前的循环索引（数字）是多少。”

NumPy：
```
ans = np.ones((3, 4), dtype=int)
```
