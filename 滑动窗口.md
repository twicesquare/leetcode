# `Counter()`
`Counter(p)` 返回一个 Counter 对象，它是字典的子类，用于统计可迭代对象中元素出现的次数。
```
from collections import Counter

p = "abcab"
p_count = Counter(p)

print(p_count)  # Counter({'a': 2, 'b': 2, 'c': 1})
print(type(p_count))  # <class 'collections.Counter'>
print(isinstance(p_count, dict))  # True (Counter 是 dict 的子类)
```
